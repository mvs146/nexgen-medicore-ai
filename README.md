## 🚀 Quick Start

### Prerequisites
- Python 3.8+ 
- 4GB+ RAM
- Internet connection for database access

### Installation & Setup
```bash
# Clone the repository
git clone https://github.com/your-username/nexgen-medicore-ai.git
cd nexgen-medicore-ai

# Create virtual environment (recommended)
python -m venv medicoreai_env
source medicoreai_env/bin/activate  # Linux/Mac
# OR: medicoreai_env\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Run the platform
python nexgen_medicore_backend.py

# Access the platform
# Open browser: http://localhost:5000
# Login: admin / [check console for generated password]
```

### 🐳 Docker Quick Start
```bash
# Build and run with Docker
docker build -t nexgen-medicore .
docker run -p 5000:5000 nexgen-medicore

# Access at http://localhost:5000
```
